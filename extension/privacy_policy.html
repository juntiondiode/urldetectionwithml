Privacy Policy for ML Phishing Detector

Last updated: March 23, 2025

Information Collection and Use

The ML Phishing Detector extension collects and processes the following information to provide phishing detection services:

  • URLs of websites you visit (for real-time phishing detection)
  • Basic URL features (length, domain structure, etc.)

Data Processing

All URL analysis is performed in real-time through our secure API. We:

  • Do NOT store any URLs or browsing history
  • Do NOT collect personal information
  • Do NOT track your browsing behavior
  • Do NOT share any data with third parties

Technical Data

The extension processes:

  • URL structure and patterns
  • Domain information
  • SSL/HTTPS status
  • DNS resolution status

Data Security

We implement security measures to protect your data:

  • All API communications use HTTPS encryption
  • No personal data is stored
  • Real-time processing without data retention

Updates to This Policy

We may update this privacy policy from time to time. We will notify users of any material changes through the extension.
